## Git

아래 순서대로 작업해주세요.

git 설정

```bash
git clone https://github.com/SayingPotatos/calculator.git # 저장소 복제
git checkout -b 요구사항명 # 브랜치 생성
```

작업물 저장

```bash
git status # 현재상태 확인
git add . # 작업한 결과물 전부 stage상태로 변경
git commit -m 'fix: 작업내용' # 커밋
git push # 원격저장소에 올리기
```

## 커밋 컨벤션

커밋은 다음 양식으로 해주세요.

fix: 수정

feat: 기능추가

chore: 기타 간단한 작업
